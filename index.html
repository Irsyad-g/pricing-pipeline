eSIM Pricing Automation Pipeline

End-to-end Python pipeline replacing a fully manual pricing process for a multi-country telecom eSIM reseller. Computes behaviour-adjusted cost estimates using a P50/P75 usage blend with 5-tier interpolation fallback (EXACT → REGION → GLOBAL → NEAREST → CURVE), applies a self-calibrating Quota Feasibility Factor, then prices via a log-linear model calibrated from competitor data (5.7% error). Isotonic regression smoothing enforces monotonic price curves. Publishes 10,800+ SKUs to Google Sheets automatically each run.

Multi-Channel Margin Reconciliation Engine

Revenue intelligence engine that reconciles order exports from Shopee, Tokopedia, and Shopify against actual COGS from the pricing pipeline database. Handles SKU normalisation across platforms, corrects Shopify's order-total pricing to per-unit via pre-dedup item counts, and classifies each ICCID as profitable (BAGUS ≥30%), breakeven (NORMAL), or loss-making (RUGI). Produces a formula-driven Excel workbook auditable by non-technical stakeholders, a Google Sheets live dashboard, and a persistent PostgreSQL backlog for unresolved orders with auto-resolution on subsequent runs.
