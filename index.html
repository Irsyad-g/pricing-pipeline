Projects

Selected work

01
eSIM Pricing Automation Pipeline

End-to-end Python pipeline that replaces a fully manual pricing process. Ingests raw daily usage and subscription XLSX files, computes behaviour-adjusted cost estimates per SKU using a 5-tier interpolation model, and publishes 10,800+ prices to Google Sheets automatically each run. Includes a market-calibrated log-linear pricing formula (5.7% error vs. benchmark) and isotonic regression smoothing.

Python Pandas scikit-learn SciPy Google Sheets API Statistical Modelling
Live
02
ICCID Checker — LAN Web App

Local network web app built with Flask and PostgreSQL for the operations team. Allows team members to look up any SIM card (ICCID) and instantly see usage, quota, roaming country breakdown, and proportional cost — with a McKinsey-style card UI. Supports bulk lookup (up to 50 ICCIDs at once) via comma/newline input.

Flask PostgreSQL psycopg2 HTML/CSS LAN Deployment
Live
03
Business Intelligence Export — 11-Sheet BI Report

Automated BI export module that generates 11 analysis sheets from PostgreSQL data — covering executive summary, country breakdown, duration/quota segmentation, SKU profitability with margin analysis, channel performance, cross-tab matrices, and churn/anomaly detection with estimated waste per ICCID. Outputs both Looker Studio-ready CSV files and an internal Excel workbook.

Pandas PostgreSQL Looker Studio Excel / openpyxl Business Intelligence
